TIE Construct

Available Since: TIE (0.9.0, 1.0.0, 1.1.0, 1.2.0, 1.3.0)

TIE's Construct Function

Syntax object new TIE ( object $owner, array $config )
Parameters
$owner

The Owner object.

$config

An array of config we want to persist to $suit->tie->config.

Return Values

The created TIE object.

Examples
Basic Usage
Code <?php
$config = array
(
    'cookie' => array
    (
        'domain' => '',
        'length' => 2678400,
        'path' => '',
        'prefix' => ''
    ),
    'flag' => array
    (
        'debug' => false
    ),
    'navigation' => array
    (
        'array' => $_GET,
        'list' => 10,
        'pages' => 2,
        'refresh' => 2
    ),
    'templates' => array
    (
        'badrequest' => 'tie/badrequest',
        'delete' => 'tie/delete',
        'form' => 'tie/form',
        'entries' => 'tie/entries',
        'pagelink' => 'tie/pagelink',
        'redirect' => 'tie/redirect',
        'xml' => 'tie/xml'
    )
);
$suit->tie = new TIE($suit, $config); //Result: Creates the TIE object
?>
